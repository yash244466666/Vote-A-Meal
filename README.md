# Vote-A-Meal

## Start Backend With Docker 

```bash
$ docker-compose up --build
```

## Start Backend without Docker 

```bash
## cd into backend
$ cd backend/

# install packages
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Start Frontend 

```bash
# cd in frontend
$ cd frontend/

# Then, run the frontend server:
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
